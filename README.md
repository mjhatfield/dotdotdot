dotdotdot is a VS Code extension that helps users easily manage the usage of packaged Copilot "Thinking Phrase" packs.

When enabled, dotdotdot applies phrase packs to VS Code's built-in thinking phrase setting in replace mode and ensures phrases are branded with the dotdotdot suffix.

## Configuration

User-facing settings:

- dotdotdot.enabled
  - Type: boolean
  - I want DOTDOTDOT to manage the Thinking Phrases for my Chat Agent.
- dotdotdot.phrasePack.the-sims
  - Type: boolean
  - Motherlode Pack (The Sims): loading messages from various The Sims games.
- dotdotdot.phrasePack.dwarf-fortress
  - Type: boolean
  - Hidden Fun Stuff (HFS) Pack (Dwarf Fortress): phrases inspired by patch notes and inside jokes.
